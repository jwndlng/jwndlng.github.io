Hi, I'm Jan!

I am a Senior Security Engineer based in Zurich, currently focused on securing the decentralized web at the DFINITY Foundation.

I see myself as a software engineer specialized in security, with a focus on infrastructure, cloud, and detection engineering. To me, security is fundamentally an engineering
challenge that requires the same approach, testing, and standards as any other software product.

This aligns with my core belief that security must be accessible. It should never be a "black box"; instead, it should be a transparent, integrated, and well-engineered part of
the development lifecycle.

This blog is about the intersection of software engineering and security operations. I will use this space to document my journey in learning and applying new technologies, building new security tools, and sharing the engineering challenges I encounter along the way.
